# Quick Add — 3 Taps to Schedule

Adding something to your family's calendar should be faster than scribbling on a sticky note. With Quick Add, it is. Tap the add button, type a title, pick a time, and you are done. Three taps from thought to schedule — no forms, no required fields, no friction.

Most calendar apps bury the creation flow behind a wall of options: location, notes, attendees, reminders, colour coding. Nesta flips that around. The only thing you need is a title and a time. Everything else is optional, tucked away for the rare occasion you need it. The result is a creation flow so fast that you will actually use it in the middle of a conversation, while cooking dinner, or waiting at the school gate.

## Key Features

- Three-tap creation: add button, title, time — done
- Smart defaults for family member assignment based on context
- Optional fields (recurrence, notes, category) available but never required
- Inline time picker that does not take over the whole screen
- Immediately visible on every paired device after creation

## How It Works

Tap the floating add button on any view — day, week, or month. A minimal input sheet slides up from the bottom with just two fields: a title and a time. Type "Emma swimming" and pick 3:30pm. Tap save. That is it — the event appears on the timeline instantly, colour-coded to Emma, synced to every device in the house.

If you want more detail, swipe the sheet up to reveal optional fields: assign to a different family member, add a note, set a recurrence pattern, or choose a category. But these extras never stand between you and getting something on the calendar. The philosophy is simple: capture first, refine later.
