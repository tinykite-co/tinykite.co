        ## 8. App-Specific Disclosures

        ### Kite AI (iOS / iPadOS / macOS / Android)

        Kite AI is a multi-provider AI chat client. We do not run any backend servers that process your conversations. The following describes data handling specific to this app.

        *   **Conversations** are stored locally on your device using the platform's standard encrypted storage (Hive on iOS/Android Keychain-protected containers). They are not transmitted to TinyKite at any point.
        *   **API keys** for third-party model providers (Anthropic, OpenAI, Google, OpenRouter, Together, Groq, etc.) that you supply in Settings are stored exclusively in your device's secure enclave (iOS Keychain / Android Keystore). They are never sent anywhere except directly to the provider's API endpoint when you send a message.
        *   **Chat content sent to model providers** travels directly from your device to the provider's API over HTTPS. We do not see, log, or proxy these requests. Each provider's privacy policy governs how they handle your prompts.
        *   **Voice dictation** is processed on-device using Apple's Speech framework / Android's local recognizer. Audio is not uploaded to TinyKite.
        *   **Optional GitHub sync**: if you connect a GitHub account in Settings, the app creates a private repository on your account and syncs your conversation history there as an append-only log. The repository is owned by you and stored in your GitHub account, not TinyKite. Your API keys are never written to this repository. You can disconnect or delete the repository at any time without affecting in-app data.
        *   **Telemetry / analytics**: none. The app makes no requests to TinyKite-controlled servers.
        *   **Permissions requested**: camera and photo library (to attach images to messages), microphone and speech recognition (for dictation). All permissions are optional — denying them only disables the corresponding feature.

        For App Store privacy nutrition label purposes: data linked to you = none collected by TinyKite. Data not linked to you = none collected by TinyKite. Data collected by third-party model providers is governed by their respective policies.
